## Commands

| Command            | Description                                      |
| :----------------- | :----------------------------------------------- |
| `pnpm dev`         | Start local dev server                           |
| `pnpm dev:search`  | Build, index with Pagefind, then start dev server|
| `pnpm build`       | Production build with Pagefind search indexing    |
| `pnpm preview`     | Preview the production build locally              |

## Deployment

Deployed on [Vercel](https://vercel.com). Pushes to `main` trigger automatic production deploys.
